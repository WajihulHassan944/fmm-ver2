# Fantasy MMAdness V2 Frontend

Next.js + TypeScript frontend for the upgraded Fantasy MMAdness platform.

## Included pages

- Public homepage, FAQ/help center, How to Play, contests, contest detail
- Player dashboard, prediction builder, live fight room, wallet/rewards, results/score breakdown
- Affiliate dashboard, contest builder, payout center, social automation
- Sponsor dashboard, create campaign, audience/inventory
- Admin dashboard, moderation operations, fraud/KYC center

## Quick start

```bash
cp .env.example .env.local
npm install
npm run dev
```

Frontend runs on `http://localhost:3000` and expects the backend at `http://localhost:4000/api`.
