# Mobile Homepage Client Reference V3

Implemented the supplied phone homepage reference as a mobile-only homepage redesign.

## Scope

- Premium fixed mobile chrome with hamburger drawer, FM wallet, add-token action, and notification/account shortcut.
- Red/blue combat-arena hero with dynamic fighter imagery, branded crown lockup, three-line campaign headline, CTA, and supporting copy.
- Four compact feature benefits.
- Player journey and Fight IQ progression preview.
- Five sport cards using category-specific fighter artwork and category colors.
- Category selection updates the live fight, analytics panel, and selected-category fight rail.
- Live fight faceoff and community prediction visualization.
- Dedicated selected-category fights section directly below the category/live area.
- Four compact platform metrics.
- Six-action mobile bottom navigation with emphasized prediction CTA.
- Desktop homepage remains on the existing desktop implementation.
