# Phase 5 Admin Split + SEO/Guides Frontend Update

## Summary

This update is additive and preserves existing functionality/assets.

## Changes

- `/administration/fights` now shows regular Match records only.
- `/administration/ShadowFightsLibrary` now shows Shadow fight/template records only.
- Scoring remains available for both regular fights and shadow fights.
- Shadow library now includes Score and Scores actions.
- Static SEO pages were restyled with a cleaner premium layout inspired by the homepage/user/affiliate theme.
- SEO landing pages now use simpler premium panels, cleaner structure, and lighter visual hierarchy.
- Guide pages were restructured into a concise premium guide center.
- Old long guide screenshot sections were replaced with action-based cards and scoring reminders.
- Existing fonts, assets, routes, and automation UI are preserved.
