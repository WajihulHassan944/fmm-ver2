# Phase 3 Frontend — Daily SEO/Social/Dashboard UX Update

## Scope
This frontend update responds to the latest client review around the public homepage, user dashboard freshness, and daily swarm automation controls.

## Updates
- Moved the fight-night promotional block off the public homepage and into the authenticated user dashboard.
- Made the user dashboard show fresh fight opportunities prominently.
- Added user dashboard CTA buttons for Play Tonight, Fight Calendar, Pro Wrestling, and Fight News.
- Kept the Play Tonight CTA consistent with the existing red Fantasy MMAdness theme.
- Improved dashboard fight opportunity ordering using featured/live/tonight/recent sorting.
- Added admin Swarm Command Center controls for daily SEO, daily social drafts, daily calendar refresh, weekly reporting, and the 1000-user growth plan.
- Added visible X/Twitter, Instagram, and Facebook social draft support in the admin swarm interface.
- Kept real social publishing review-controlled and draft-safe until credentials and final approval are configured.
