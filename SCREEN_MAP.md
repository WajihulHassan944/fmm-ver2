# Screen Map

## Public
- `/` Homepage with active contests, recent winners, leaderboard, trust proof
- `/faq` FAQ & Help Center
- `/how-to-play` Rules and scoring guide
- `/contests` Contest listing
- `/contests/[id]` Contest detail

## Player
- `/dashboard` Player dashboard
- `/prediction/[contestId]` Prediction builder
- `/live/[contestId]` Live fight room with Socket.IO hooks
- `/results/[contestId]` Results & score breakdown
- `/wallet` Wallet, rewards, transaction ledger, payout methods

## Affiliate
- `/affiliate` Affiliate dashboard
- `/affiliate/builder` Affiliate contest builder
- `/affiliate/payouts` Affiliate earnings and payout center
- `/affiliate/social-automation` Fighter/social automation kit

## Sponsor
- `/sponsor` Sponsor overview
- `/sponsor/create-campaign` Create sponsorship campaign
- `/sponsor/audience` Audience insights and placement inventory

## Admin
- `/admin` Super admin dashboard
- `/admin/moderation` Contest moderation and payout operations
- `/admin/fraud` Verification and fraud review center
