# Phone Homepage Roadmap V7

- Five combat categories now fit in one mobile viewport and display live API-backed counts.
- Category selection updates the fight feed in place without forced page scrolling.
- Featured upcoming fight posters are clickable and open the existing premium details flow.
- Fight cards remain the only dynamic fighter-image surfaces; non-fight modules use static design artwork or iconography.
- Community prediction percentages are calculated from available fight prediction records rather than hard-coded values.
- Open fight, player, prize-pool, wallet, XP, and leaderboard values use current application data.
- Homepage fight, promoted-poster, leaderboard, and wrestling feeds refresh every 60 seconds while preserving the last successful state during transient refresh errors.
- Added clickable roadmap destinations for open events, apparel, blogs, and affiliates.
- Added an animated affiliate availability bubble showing the current number of open matches.
- Bottom navigation now includes a direct Affiliate destination.
- Added `/apparel` as a valid destination for the new apparel shortcut.
